- requirements.txt
- Procfile
- railway.json

## أمر التشغيل في Railway
gunicorn app:app --bind 0.0.0.0:$PORT

## ملاحظة الكويكبات
التطبيق يحاول تحميل ملف seas_18.se1 تلقائيًا عند الحاجة.
إذا لم تظهر الكويكبات، أنشئ مجلد sweph داخل المشروع وضع فيه ملف seas_18.se1.


## نسخة بايثون
تمت إضافة ملف .python-version لتثبيت Python 3.11 على Railway من أجل استقرار أعلى مع Flask وGunicorn وpyswisseph.

## طريقة الرفع
ارفع محتويات هذه الحزمة إلى Railway، أو ارفعها إلى GitHub ثم اربط المستودع بـ Railway.
أمر التشغيل محدد مسبقًا في Procfile وrailway.json:
gunicorn app:app --bind 0.0.0.0:$PORT
